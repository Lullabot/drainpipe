# Instructions for AI Agents

This is a project for integrating developer tools and CI workflows into Drupal projects.

Key technologies include:

- ddev
- task (from https://taskfile.dev/)
- Drupal
- GitHub Actions
- Bitbucket Pipelines
- Acquia hosting controlled with the `acli` command
- Pantheon hosting managed with the `terminus` command
- Tugboat for pull request previews

**ALWAYS** use an agent that is a Drupal and DevOps expert.
